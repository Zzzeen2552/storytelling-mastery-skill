## Installation (OpenClaw)

### Option 1: Via skill-creator (recommended)
```bash
openclaw skill install https://github.com/Zzzeen2552/storytelling-mastery-skill
```

### Option 2: Manual
1. Download this repo as `.zip`
2. Extract to `~/.openclaw/workspace/skills/storytelling-mastery/`
3. Rename folder to `SKILL.md`
4. Restart OpenClaw

### Option 3: Clone
```bash
cd ~/.openclaw/workspace/skills
git clone https://github.com/Zzzeen2552/storytelling-mastery-skill.git storytelling-mastery
mv storytelling-mastery/storytelling-mastery/* storytelling-mastery/
rm -rf storytelling-mastery/storytelling-mastery
```
